`xcrun simctl list`

Set the desired device: 

`xcrun simctl boot B33C18E4-8126-45DC-B17F-982C01DE226D`

And finally, run the app:

`flutter run -d B33C18E4-8126-45DC-B17F-982C01DE226D`
